# Contributing to PowerShell Radio Pro

Thank you for your interest in contributing!

## Development Setup

```bash
# Clone the repository
git clone https://github.com/your-username/radio_app_cli_powershell.git
cd radio_app_cli_powershell

# Install in development mode
pip install -e radio_ps[dev]

# Run the app
python -m radio_ps
```

## Code Style

- Use Black for formatting
- Use type hints where possible
- Keep functions small and focused

## Testing

```bash
# Run tests
pytest
```

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Issues

Use the issue templates to report bugs or request features.
